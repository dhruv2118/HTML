    What is difference between HTML and XHTML?

    1. HTML stands for Hyper Text Markup Language, and XHTML stand for Extensible Hypertext Markup Language which is a connection between HTML and XML(Extensible Markup Language).
    2. HTML is not case-sensitve so tags and attributes could be written in lower or upper case as needed, where as XHTML is case-sensitive and strict with every tags and attributes written in the code should be lower case only.
    3. HTML files can be stored with .htm,.html extensions; whereas XHTML can be stored with .xhtml,.xht or .xml extensions.
    4. HTML can add attributes without any quote and in XHTML, it is mandatory to add quotes on every declared attribute.

    What are logical and physical tags in HTML?

    There are 2 types of tags in HTML:

    1. Logical Tags:

      2. Logical tags are tags that describe the structure and meaning of the content, without specifying how it should be displayed.

      for ex. Heading tags(<h1> to <h6>) ; Paragraph tag (<p>) ; Table tag(<table>) and more...
      These tags provide meaning to the content and help screen readers to understand the content structure.

    2. Physical Tags:

      3. Physical tags are tags that describe how the content should be displayed on the screen.

      for ex. Bold and strong tag(<b> and <strong>) ; Underline tag (<u> or <ins>) and many more...
      These tags normally provide visual styling to the content.
